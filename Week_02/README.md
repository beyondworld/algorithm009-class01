## 二叉堆
  堆的调整算法
  - 入堆调整 heapfiyUp()
    - 将节点插入到堆中最末尾
    - 由末尾向上调整，找到父亲节点 parent = index / 2
      - 如果父亲节点 大于 或者小于 当前节点（根据大根堆或者小根堆判断）,交换节点。如果不满足条件则退出调整
      - 更新index = parent
    - 如果parent <= 0 则调整完毕
  - 出堆调整 heapfiyDown()
    - 将根节点出堆，将最后一个节点放到根节点
    - 从上往下调整，找到parent节点的两个孩子，和最大或者最小的节点比较，判断是否交换。
      - 如果交换，则更新parent为子节点
      - 如果不交换，则退出调整
    - 如果parent节点大于堆大小，则调整结束
